# Hook Templates
# Version: 1.0.0

## Hook Types

### 1. Mystery Hook
Start with a puzzling situation that creates curiosity.
Template: "לקוח אמר לי משהו שלא הבנתי. אבל אז..."
Best for: Complex cases with surprising twists

### 2. Contrast Hook  
Show unexpected reversal of expectations.
Template: "כולם חשבו ש-X. התברר בדיוק ההיפך."
Best for: Stories that challenge conventional wisdom

### 3. High Stakes Hook
Lead with consequences to create urgency.
Template: "היו לנו שבועיים למנוע נזק של X מיליון שקל."
Best for: Time-sensitive, high-pressure situations

### 4. Pattern Interrupt Hook
Challenge common beliefs directly.
Template: "רוב עורכי הדין עושים X. אני מגלה שהם טועים."
Best for: Thought leadership, differentiation

### 5. Insider Secret Hook
Reveal professional insight that feels exclusive.
Template: "אחרי 20 שנה בתחום, למדתי משהו שמפתיע אותי עד היום."
Best for: Wisdom-based content, credibility building

### 6. Story Hook
Start mid-action to create immediate engagement.
Template: "הטלפון צלצל ב-2 בלילה. [Client type] היה בפאניקה."
Best for: Dramatic narratives, client stories

### 7. Wisdom Hook
Lead with counterintuitive insight.
Template: "כשלקוח אומר X, הוא באמת מתכוון ל-Y. רוב עורכי הדין מפספסים את זה."
Best for: Strategic insights, pattern recognition

## Hook Generation Instructions
Given the source material, generate 3 hooks:
1. Best match for the story type
2. Alternative angle
3. Different emotional approach

Score each hook on:
- Attention Grab (stops the scroll)
- Curiosity (makes you want to read more)
- Authenticity (sounds like Eitan)
- Promise Clarity (clear what reader will learn)

Return as JSON array with hook_type, hook_text_hebrew, and scores.
